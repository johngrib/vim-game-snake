## How to play

`:VimGameSnake` to Start

h   | j   | k   | l   | c        | q
--- | --- | --- | --- | ---      | ---
←   |↓    |↑    |→    | end game | quit

## Installation

### VimPlug

Place this in your .vimrc:

> Plug 'johngrib/vim-game-snake'

Then run the following in Vim:

> :source %

> :PlugInstall
